DRINK SELECT
 ## 설치 방법

 1. 프로젝트 클론
    git clone https://github.com/soobin0910/Drink_Select.git

 2. 가상 환경 설정
    python3 -m venv venv

 3. 가상 환경 활성화
    Mac : source venv/bin/activate
    Window : venv\Scripts\activate

 4. 의존성 설치
    pip install -r requirements.txt

 5. Flask 서버 실행
    python app.py

 6. React 실행
    npm install
    npm run dev

 7. 가상 환경에서 나오기
    deactivate
